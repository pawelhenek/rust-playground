Rust playground
===

#### Prerequisites

Installed Rust (https://www.rust-lang.org/tools/install)

### Quick start
`git clone https://github.com/pawelhenek/rust-playground.git`

`cd rust-playground`

`cargo build`

`cargo test`

`cargo run`

### Locally verified with

`rustc --version`

`rustc 1.76.0 (07dca489a 2024-02-04)`
